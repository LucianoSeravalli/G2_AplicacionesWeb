Bienvenido al Dashboard de Administración

Desde este panel podrás gestionar de forma centralizada las categorías, productos y transacciones de Luk Sport Center. Este apartado está diseñado para facilitar el control del inventario, la organización del catálogo y el seguimiento de las operaciones realizadas dentro del sistema.

¿Qué puedes hacer aquí?

  • Administrar categorías del catálogo
  • Gestionar productos e inventario
  • Consultar transacciones del sistema
  • Mantener organizado el contenido comercial
